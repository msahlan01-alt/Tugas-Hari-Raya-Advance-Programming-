# Advanced Programming Assignment

This repository contains the implementation of five programming tasks focused on Object-Oriented Programming (OOP) concepts in Java, including inheritance and polymorphism.

## 📁 Project Structure

* Question1 → Solution for Question 1
* Question2 → Solution for Question 2
* Question3 → Solution for Question 3
* Question4 → Solution for Question 4
* Question5 → Solution for Question 5

Each folder contains:

* Java source code file
* Output screenshots as proof of execution

## 🛠️ Technologies Used

* Java
* OOP Concepts (Inheritance, Polymorphism, Overriding)
* Data Structures (Stack, Queue, HashMap)

## ▶️ How to Run

1. Open the desired question folder
2. Compile the Java file:

   ```
   javac Qx.java
   ```
3. Run the program:

   ```
   java Qx
   ```
